# ADRs

Registro de decisiones de arquitectura del proyecto.

## Convención

- Un ADR por decisión significativa.
- Mantener estado del ADR (Propuesto, Aprobado, Reemplazado).
- Incluir alternativas consideradas y consecuencias.

## Índice recomendado

- ADR 001 ...
- ADR 002 ...
- ...

Agregar nuevos ADRs siguiendo la plantilla en `ADRs/ADR-TEMPLATE.md`.
